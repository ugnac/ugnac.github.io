## 基础语法

下列元素语法是所有 Markdown 引擎都支持的，可以放心使用。

| 元素     | Markdown 语法                                                          |
| -------- | ---------------------------------------------------------------------- |
| 标题     | `# 一级标题`、`## 二级标题`                                        |
| 加粗     | `**粗体文本**`                                                       |
| 强调     | `*斜体文本*`                                                         |
| 块引用   | `> 引用内容`                                                         |
| 有序列表 | `1. 列表项`                                                          |
| 无序列表 | `- 列表项`                                                           |
| 代码     | `代码`                                                               |
| 分隔线   | `---`                                                                |
| 超链接   | `[B3log](https://b3log.org)`                                         |
| 图片     | `![B3log Logo](https://static.b3log.org/images/brand/b3log-128.png)` |

## 扩展语法

下列元素语法并不是所有 Markdown 引擎都支持，使用前请进行确认。

| 元素     | Markdown 语法                     |
| -------- | --------------------------------- |
| 表格     |                                   |
| 代码块   | ```                               |
| 脚注     | `这里是一个脚注引用[^1]`        |
| 标题 ID  | `### 这是一个标题 {#custom-id}` |
| 删除线   | `~~地球是平的。~~`              |
| 任务列表 | `- [x] 待办事项一`              |
